Training Bot RAG Hub は通常のWebアプリではなく、

- 金融システム
- RAGシステム
- LLMシステム
- 外部データ連携システム

の複合体です。

そのため OWASP ASVS + OWASP API Top10 + OWASP LLM Top10 を統合した試験が必要です。要件定義で定義した Prompt Injection、Secret Masking、Order Permission、Provider Adapter、外部LLM送信制御を重点的に検証します。

# **2. セキュリティ試験方針**

## **目的**

以下を保証する。

```text
RAGから注文できない

秘密情報が漏れない

Prompt Injectionで乗っ取られない

外部LLMへ危険情報を送らない

監査可能である
```
