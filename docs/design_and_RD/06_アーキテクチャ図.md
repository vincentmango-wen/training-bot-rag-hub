## Query シーケンス
```mermaid
sequenceDiagram
  participant Bot as Training Bot
  participant API as rag-api
  participant Orch as rag-orchestrator
  participant Ret as rag-retriever
  participant DB as PostgreSQL / pgvector
  participant LLM as LLM Provider Adapter
  participant Guard as rag-guardrail
  participant Audit as rag-audit-logger

  Bot->>API: POST /api/v1/rag/bot-context
  API->>API: JWT認証 / Request Validation
  API->>Orch: Bot Context Query
  Orch->>Ret: 検索要求
  Ret->>DB: Semantic Search + Metadata Filter
  DB-->>Ret: Relevant Chunks
  Ret-->>Orch: Retrieval Results
  Orch->>LLM: Structured Answer Generation
  LLM-->>Orch: Generated Response
  Orch->>Guard: Schema Validation / 禁止表現検知
  Guard->>Guard: order_permission=false 強制
  Guard->>Audit: Query / Source / Response / Provider Usage保存
  Audit->>DB: Audit Log保存
  Guard-->>API: Validated Response
  API-->>Bot: Context Response
```
